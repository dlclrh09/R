1. if, else문

if (조건식) {

  조건식이 참일 경우 실행하는 명령

} 

else {

  조건식이 거짓일 경우 실행하는 명령

}

. if, else if, else문

else if 를 이용하면 여러개의 조건식을 넣을 수 있다. 

​

if (조건식1) {

  조건식1이 참일 경우 실행하는 명령

} 

else if (조건식2) {

  조건식1이 거짓이고 조건식2가 참일 경우 실행하는 명령

} 

else {

  조건식이 전부 거짓일 경우 실행하는 명령

}
